## Login & Account Issues

### Can't Log In

**Problem:** "Invalid email or password"

**Solutions:**
1. **Reset password:**
   - Click "Forgot Password"
   - Check email (including spam)
   - Create new password
   
2. **Email typo:**
   - Check spelling carefully
   - Try alternate email

3. **Account suspended:**
   - Check email for notices
   - Contact support@craudiovizai.com

**Problem:** Not receiving password reset email

**Solutions:**
1. Check spam/junk folder
2. Wait 10 minutes (may be delayed)
3. Try different email
4. Contact support with your account email

### Two-Factor Authentication Issues

**Problem:** Lost 2FA device

**Solutions:**
1. Use backup codes (saved during setup)
2. Contact support with:
   - Account email
   - Last login date
   - Recent project names
   - Payment method (last 4 digits)

**Problem:** 2FA code not working

**Solutions:**
1. Ensure device time is accurate
2. Wait for new code (changes every 30 seconds)
3. Check code expiration
4. Try backup code

## Tool & Creation Issues

### Tool Won't Load

**Problem:** Spinning wheel, blank screen

**Solutions:**
1. **Refresh page:** Press F5 or Ctrl+R
2. **Clear cache:**
   - Chrome: Ctrl+Shift+Delete
   - Choose "Cached images and files"
   - Clear last 24 hours
3. **Try incognito mode:** Test if extensions interfere
4. **Different browser:** Try Chrome, Firefox, Safari
5. **Check status:** [status.craudiovizai.com](https://status.craudiovizai.com)

### Export Failed

**Problem:** "Export failed" error

**Solutions:**
1. **Check file size:** Max 100MB per export
2. **Sufficient credits:** Ensure you have enough
3. **Wait and retry:** Server may be busy
4. **Try different format:** PNG instead of SVG
5. **Simplify project:** Remove unnecessary elements

**Problem:** Export takes forever

**Expected Times:**
- Images (PNG/JPG): 10-30 seconds
- PDFs: 30-60 seconds
- Videos: 2-5 minutes
- Complex designs: Up to 10 minutes

**If longer:**
1. Check internet speed
2. Close other apps
3. Try smaller file size
4. Export in sections

### Upload Failed

**Problem:** "Upload failed" error

**Solutions:**
1. **Check file type:** Supported formats:
   - Images: PNG, JPG, GIF, SVG
   - Documents: PDF, DOCX, TXT
   - Video: MP4, MOV
   - Max size: 50MB

2. **Reduce file size:**
   - Compress images
   - Use online compressor
   - Split large files

3. **Rename file:**
   - Remove special characters
   - Use only letters, numbers, dash
   - Example: `my-image-2025.png`

### Lost Work / Project Not Saving

**Problem:** Changes disappeared

**Solutions:**
1. **Check auto-save:** Look for "Saving..." indicator
2. **Manual save:** Click Save button
3. **Check internet:** Connection required for save
4. **Recent versions:** View â†’ Version History
5. **Recovery:** Support can restore (24-hour window)

**Prevention:**
- Save frequently (Ctrl+S)
- Check "Last saved" timestamp
- Download local backup
- Use stable internet

## Error Messages Explained

### "Rate limit exceeded"
**Meaning:** Too many requests too fast  
**Solution:** Wait 1 minute, try again

### "Insufficient credits"
**Meaning:** Not enough credits for action  
**Solution:** Buy more credits or use simpler tool

### "Server error 500"
**Meaning:** Our server issue  
**Solution:** Wait 5 minutes, try again. If persists, contact support

### "Unauthorized"
**Meaning:** Not logged in or session expired  
**Solution:** Log out and log back in

### "Forbidden"
**Meaning:** Don't have permission  
**Solution:** Check account type or contact support

### "Not found"
**Meaning:** File or page doesn't exist  
**Solution:** Check URL or search for item
