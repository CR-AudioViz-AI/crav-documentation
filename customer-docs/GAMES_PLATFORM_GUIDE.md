## Troubleshooting

### Common Issues

**Game Won't Load:**
- Refresh page
- Clear browser cache
- Try different browser
- Check internet connection
- Disable ad blockers

**Lag/Slowness:**
- Close other programs
- Lower graphics quality (if available)
- Use wired internet
- Update browser
- Restart device

**Lost Progress:**
- Check if auto-save enabled (Pro/Elite only)
- Contact support with details
- May be recoverable
- Free tier: No progress save

**Controls Not Working:**
- Check keyboard/mouse connection
- Try different input device
- Review game controls
- Clear browser cache
- Restart game
