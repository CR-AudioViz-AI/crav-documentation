## Common Issues & Solutions

### Can't Log In
1. Check email spelling
2. Try password reset
3. Check spam for reset email
4. Verify account exists
5. Check for suspension

### Tool Not Working
1. Clear browser cache
2. Try incognito mode
3. Check status page
4. Verify credit balance
5. Try different browser

### Payment Failed
1. Verify card details
2. Check sufficient funds
3. Try PayPal alternative
4. Contact bank
5. Manual payment option

### Export Not Working
1. Check file size (<100MB)
2. Verify credits available
3. Try different format
4. Wait and retry
5. Escalate if persists

### Missing Credits
1. Refresh page
2. Check transactions
3. Verify payment processed
4. Wait 5 minutes
5. Manual credit adjustment

## Policy Quick Reference

### Refund Policy
- **Automatic:** System errors, failed exports
- **Manual approval:** Changed mind (within 30 days)
- **No refund:** Used credits, completed work

### Account Termination
- **Reasons:** TOS violations, illegal activity, abuse
- **Process:** Warning first, then suspension
- **Appeal:** 30 days to resolve

### Data Requests
- **Export:** Provide within 7 days
- **Delete:** Complete within 30 days
- **GDPR:** Priority handling

## Emergency Procedures

### System Down
1. Confirm on status page
2. Alert #incidents channel
3. Use downtime template
4. Update every 30 minutes
5. Notify when resolved

### Security Breach
1. Alert security team immediately
2. Do NOT discuss publicly
3. Follow security protocol
4. Document everything
5. Management notification

### Data Loss
1. Alert engineering immediately
2. Reassure customer
3. Attempt recovery
4. Management approval for compensation
5. Document incident
