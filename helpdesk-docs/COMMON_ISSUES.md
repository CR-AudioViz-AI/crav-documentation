# COMMON ISSUES & SOLUTIONS

## Issue 1: App Won't Load
**Symptoms:** Clicking app shows blank page or error
**Solution:**
1. Refresh page (Ctrl+R)
2. Clear browser cache
3. Try different browser
4. Contact support if persists

## Issue 2: Credits Not Deducted
**Symptoms:** Used app but credits unchanged
**Solution:**
- This is a system error
- Credits auto-refunded
- Use app again
- Email us if repeats

## Issue 3: Payment Failed
**Symptoms:** Card declined or payment error
**Solution:**
1. Verify card details correct
2. Check sufficient funds
3. Try PayPal alternative
4. Contact your bank
5. Email: billing@craudiovizai.com

## Issue 4: Forgot Password
**Solution:**
1. Click "Forgot Password"
2. Enter email
3. Check email for reset link
4. Create new password
5. Email support if no email received

## Issue 5: Javari Not Responding
**Symptoms:** Chat doesn't reply
**Solution:**
1. Refresh page
2. Check internet connection
3. Try different browser
4. Wait 1 minute and retry
5. Report to support

## Contact Support
- Email: support@craudiovizai.com
- Response Time: 24-48 hours (Free), 2 hours (Premium)
- Include: Account email, issue description, screenshots
