# SUPPORT TEAM PROCEDURES

## Ticket Classification

### Priority Levels
- **P0 (Critical):** System down, payments broken
- **P1 (High):** Feature broken, credits lost
- **P2 (Medium):** Minor bug, feature request
- **P3 (Low):** Question, documentation

### Response Times
- Free users: 24-48 hours
- Basic users: 12-24 hours
- Pro users: 4-8 hours
- Premium users: 2 hours
- Enterprise: 1 hour (with SLA)

## Standard Responses

### Credit Refund
1. Verify error in logs
2. Refund credits immediately
3. Log incident
4. Notify dev team if system issue

### Account Issues
1. Verify user identity
2. Check account status
3. Resolve issue
4. Document resolution

### Escalation
Escalate to dev team if:
- System-wide issue
- Bug confirmed
- Feature broken
- Security concern
